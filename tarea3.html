Lenguaje de Progrmacion

Un lenguaje de programación es un conjunto de reglas y símbolos que se utilizan para escribir programas de computadora. Estos lenguajes permiten a los programadores comunicarse con las computadoras, dándoles instrucciones para realizar tareas específicas. Cada lenguaje de programación tiene su propia sintaxis y semántica, que determinan cómo se escriben y se interpretan las instrucciones

Clasificación de lo lenguajes de alto nivel

  • Lenguajes de Proposito general:No estan centrados en un area o tarea determinada, ejemplos;C,JAVA o Delphi.
  • Lenguajes de proposito Especifico :Como su nombre lo dice fueron diseñados para tareas especificas y cuentan con caracteristicas especificas y un alto nivel de abastraccion un ejemplo claro es SQL para la progrmacion y gestion de bases de datos.
  • Lenguajes impertativos: Utilizan instrucciones que especifican de forma clara y concisa las órdenes a ejecutar por el programa. Entre este tipo de lenguajes imperativos encontramos el Fortran,C++ y Pascal.
  • Lenguajes declarativos: En este tipo de lenguajes el código expresa lo que debe realizar un programa con una secuencia de acciones determinada. Como ejemplos de este tipo de lenguajes tenemos el SQL para bases de datos o el lenguaje de marcado HTML.
  • Lenguajes funcionales:Son lenguajes que contienen una serie de funciones ya definidas que son llamadas para realizar las distintas acciones de un programa. El Lisp o el Scheme son dos lenguajes funcionales.
  • Lenguajes logicos:Son lenguajes de programación que utilizan la lógica matemática a la hora de codificar un programa. Un claro ejemplo de este tipo de lenguaje de alto nivel lógico es Prolog, que se usa de forma habitual en el desarrollo de aplicaciones de inteligencia artificial
  • Lenguajes Orientados a Objetos: En este tipo de programación los distintos elementos son considerados como objetos, con sus propiedades y atributos propios. Utilizan sistemas de clases para definir estos objetos y la forma en que se relacionan con otros. Ejemplos como:C#,JAVA

Ejemplos de lenguajes:

  • Interpretados:PHP, Ruby, Python y JavaScript.
  • Compilados:C, C++, Java, Go y Rust
  • Orientados a Objetos:C++, Objective C, Java, Ruby, Visual Basic, Visual C Sharp, Simula, Perl, TypeScript, Smalltalk, PHP o Python.
  • Funcionales:Java,PHP,Ruby, Python,Elixir,Kotling,Haskell,Erlang.
